# Mada Kasasi - Developer Portfolio

A modern, responsive developer portfolio built with Next.js 15, showcasing expertise in systems engineering, AI, and cutting-edge technologies.

## 🚀 Features

- **Modern Design**: Vercel-inspired minimalist design with dark theme
- **Responsive**: Optimized for mobile, tablet, and desktop
- **Performance**: Built with Next.js 15 and optimized for speed
- **Animations**: Smooth animations with Framer Motion
- **SEO Optimized**: Meta tags and structured data
- **Type Safe**: Full TypeScript implementation
- **Accessible**: WCAG compliant design

## 🛠️ Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Font**: Inter (Google Fonts)

## 📱 Sections

- **Hero**: Animated introduction with particle background
- **Projects**: Filterable project showcase with tech stack
- **About**: Professional journey timeline and skills
- **Contact**: Contact form with social links

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/madakasasi/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Project Structure

```
app/
├── components/          # React components
├── data/               # Static data (projects, skills, timeline)
├── lib/                # Utility functions
├── types/              # TypeScript type definitions
├── globals.css         # Global styles
├── layout.tsx          # Root layout
└── page.tsx            # Home page
```

## 🎨 Customization

1. **Personal Information**: Update data in `app/data/` files
2. **Styling**: Modify `tailwind.config.ts` for custom colors/themes
3. **Content**: Edit components in `app/components/`
4. **SEO**: Update metadata in `app/layout.tsx`
